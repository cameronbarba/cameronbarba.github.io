Sleep-Easy

January 2019

A simple web page that provides basic information to potential guests. The site is responsive according to content breaks and provides cross-browser compatibility.

view site

Problem:

The budget was extremely tight for this project. We wanted to at least provide a space for the rental home to exist online that we could design and code in 24 hours.

Solution:

We decided that a minimal and responsive static web page would be the best option given our constraints. We also decided against any styling frameworks to avoid any unneeded bloat to our small site.

Although time was a constraint, we chose to roll custom CSS to minimize file size and increase performance

We did simple wireframes to get an overall idea of the site architecture and performed image manipulation in Photoshop to ensure all photos fit an appropriate pallete as they were the focus of the site. Most designing was done in the browser
